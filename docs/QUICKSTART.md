# Quick Start

## 1. Install dependencies

```bash
npm install
```

## 2. Prepare environment file

```bash
cp .env.example .env
```

Then adjust the values inside `.env` as needed for your local environment.

## 3. Start the project

```bash
npm start
```

## 4. Open API docs

If Swagger is enabled in the current source, it is typically exposed at:

```text
http://localhost:PORT/docs
```
